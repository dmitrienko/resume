(выберите в меню Ваш браузер)

  • Firefox
  • Opera
  • Google Chrome
  • Safari

Internet Explorer

  1. В меню Tools (Сервис) выберите пункт Internet Options (Свойства обозревателя).
       

  2. Откройте вкладку Security (Безопасность) и выберите зону Интернет (Internet). Нажмите на кнопку Custom level (Другой).



  3. В разделе Scripting (Сценарии), в подразделах Active scripting (Активные сценарии) и Scrtpting of Java applets (Выполнять сценарии приложений Java) выберите пункты Enable (Разрешить).



  4. Нажмите «ОК». Перезагрузите страницу, для этого нажмите кнопку F5.

Mozilla Firefox

  1. В меню Tools (Инструменты) выберите пункт Options (Настройки).
       

  2. Перейдите в раздел Content (Содержимое) и установите флажки рядом с Enable JavaScript (Использовать JavaScript) и Enable Java (Использовать Java).



  3. Нажмите «ОК». Перезагрузите страницу, для этого нажмите кнопку F5.

Opera

  1. В меню Tools (Инструменты) выберите пункт Preferences (Настройки).
       

  2. Перейдите на вкладку Advanced (Дополнительно). Выберите пункт меню Content (Содержимое) и установите флажки рядом с Enable JavaScript и Enable Java.



  3. Нажмите «ОК». Перезагрузите страницу, для этого нажмите кнопку F5.

Google Chrome

  1. Нажмите на значок «гаечный ключ», расположенный справа сверху;
  2. Перейдите в раздел «Параметры»;
  3. Перейдите на вкладку «Расширенные»;
  4. Нажмите «Настройки содержания..»;
  5. В меню слева выберите «JavaScript»;
  6. Установите галочку напротив «Разрешить всем сайтам использовать JavaScript»;
  7. Нажмите «Закрыть». Перезагрузите страницу, для этого нажмите кнопку F5.

Safari

  1. Выберите пункт меню «Правка» или нажмите на значок «шестеренка», расположенный справа сверху, в более поздних версиях браузера;
  2. Перейдите на вкладку «Безопасность»;
  3. Установите галочку напротив «Подключить JavaScript»;
  4. Нажмите на значок «крестик» для того, чтобы сохранить изменения. Перезагрузите страницу, для этого нажмите кнопку F5.
